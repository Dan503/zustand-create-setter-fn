# Zustand Create Setter Function

## Install

npm

`npm i zustand-create-setter-fn`

pnpm

`pnpm add zustand-create-setter-fn`

yarn

`yarn add zustand-create-setter-fn`

bun

`bun add zustand-create-setter-fn`
